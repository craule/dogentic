# Security Policy

## Supported Versions

The following versions of the Dogent Agent are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of Dogent Agent seriously. If you have discovered a security vulnerability, we appreciate your help in disclosing it to us in a responsible manner.

### Disclosure Process

1.  Please **DO NOT** open a public issue on GitHub.
2.  Email our security team at **security@dogent.net**.
3.  Include a detailed description of the vulnerability, steps to reproduce, and any proof of concept (PoC) code if available.

We will acknowledge your report within 48 hours and provide a timeline for triage and resolution.

### Response Timeline

*   **Triage:** Within 48 hours of acknowledgment.
*   **Resolution:** We aim to release a patch for critical vulnerabilities as priority.
*   **Advisory:** We will publish a security advisory once the fix is deployed and verified.

Thank you for helping keep the Dogent ecosystem secure.
